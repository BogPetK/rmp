Для упрощения использования приложения “Крымские сокровища” было составлено руководство пользователя:
1.Установить приложение.
2.Запустить.
3.После этого необходимо открыть приложение, нажав на его иконку на экране устройства.
4.После открытия приложения, пользователя встречает приветственный экран. Далее необходимо нажать кнопку “Начать”, 
чтобы перейти к основному экрану приложения.
5.На главном экране, пользователь может просмотреть список достопримечательностей и список категорий, на которые они разделены. 
Так же можно видеть список самых популярных достопримечательностей. Выбрав категорию, в списке останутся только те достопримечательности,
которые относятся к ней.
6.Далее, выбрав интересующую достопримечательность, пользователь может нажать на её панель и перейти к экрану с подробной 
информацией по достопримечательности.
7.На экране достопримечательности, пользователь может ознакомиться с ей описанием, адресом, фотографиями и ценой входа.
8.После того, как пользователь ознакомился с необходимой ему информацией, он может вернуться к списку достопримечательностей, 
нажав кнопку “назад” на устройстве либо вовсе закрыть приложение.
